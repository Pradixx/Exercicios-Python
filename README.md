# 🐍 Exercícios Python

Repositório com meus exercícios e projetos desenvolvidos durante o aprendizado de Python, organizados por nível de dificuldade (Laboratório de Python).

---

## 📁 Organização

```
📂 Básico/          → Exercícios fundamentais de Python
📂 Intermediário/   → Exercícios com lógica mais elaborada e desafios
📂 Projetos/
    ├── Básico/         → Chapéu Seletor de Hogwarts
    └── Intermediário/  → Jogo da Velha (em desenvolvimento)
```

---

## 🧠 Conteúdos Abordados

**Básico**
- Variáveis e tipos de dados
- Entrada e saída de dados (`input`, `print`)
- Operadores aritméticos e lógicos
- Condicionais (`if`, `elif`, `else`)
- Laços de repetição (`for`, `range`)
- Listas e manipulação de strings

**Intermediário**
- Funções e módulos (`math`, `random`, `datetime`)
- Classificação e validação de dados
- Laços aninhados e loops inversos
- Manipulação de listas (inserção, remoção, busca)
- Desafios clássicos: **FizzBuzz**, **Bola Mágica 8**, **Montanha-Russa**

**Projetos**
- 🎩 **Chapéu Seletor de Hogwarts** — quiz interativo que classifica o usuário em uma das casas de Harry Potter com base em suas respostas
- ⭕ **Jogo da Velha** — em desenvolvimento

---

## 🛠️ Tecnologias

- **Python 3**

---

## ▶️ Como executar

1. Clone o repositório
2. Acesse a pasta do exercício desejado
3. Execute com Python:

```bash
python Exercicio1.py
```
